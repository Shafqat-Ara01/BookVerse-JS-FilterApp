#  BookVerse

BookVerse is a **responsive and interactive book library** built with pure **HTML, CSS, and JavaScript**.  
It allows users to **filter books by genre**, **search by title or author**, and **sort books by year or alphabet** — showcasing the power of JavaScript DOM manipulation.

---

##  Features
-  **Search Functionality:** Instantly find books by title or author.  
-  **Genre Filter:** Filter through genres like Fiction, Non-fiction, Mystery, and Programming.  
-  **Sorting Options:** Sort books by publishing year or alphabetically.  
-  **Dynamic Rendering:** All book cards are generated using JavaScript DOM methods.  
-  **Fully Responsive:** Adjusts beautifully across all screen sizes.

---

##Technologies Used
- HTML5 – Structure
- CSS3 – Styling and responsiveness
- JavaScript (DOM) – Logic, filtering, and rendering
